# What is "Quick Sort"?

快速排序法，是排序法中平均較快速，但也有可能因為陣列本身排序不適合此種排序法，而需要O(n^2)時間。 排序方式是選定一個數為基準點(pivot,通常為中間值)，接著從第一個數開始依序跟基準點比大小，小的放左邊，大的放右邊，接著左右兩邊的陣列也做上述相同的步驟，選一個基準點、比大小，重複進行。
